#  Sistema de Biblioteca - Python & SQL Server

Sistema de gestión de biblioteca desarrollado en Python utilizando SQL Server como base de datos.

##  Funcionalidades

- Login con usuarios y roles (admin, bibliotecario)
- Gestión de libros (CRUD)
- Gestión de lectores
- Préstamos y devoluciones
- Validaciones de datos
- Generación automática de códigos
- Contraseñas cifradas (hash)

##  Tecnologías

- Python
- SQL Server
- pyodbc

##  Cómo ejecutar

1. Instalar dependencias:

pip install pyodbc

2. Configurar conexión en main.py

3. Ejecutar:

python main.py
