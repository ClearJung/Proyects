# Personal Expense Tracker (Python)

Sistema en consola desarrollado en Python que permite registrar, editar y eliminar gastos personales, clasificarlos por categorías, calcular totales y exportar la información a CSV.

## Funcionalidades

- Registrar gasto
- Mostrar gastos
- Editar gasto
- Eliminar gasto
- Total gastado
- Total por categoría
- Exportar a CSV
- Categorías predefinidas
- Persistencia con JSON

## Tecnologías

- Python 3
- JSON
- CSV

## Ejecución

python main.py
